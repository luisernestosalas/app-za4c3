# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es una herramienta simple para gestionar tus clientes y las notas sobre ellos. Te ayuda a:
- Guardar información de contacto
- Escribir notas sobre cada cliente
- Organizar toda tu información en un solo lugar

## Cómo acceder

**Dirección web:** https://app-za4c3-md8yj1kee-luisernestosalas-2775s-projects.vercel.app

Abre esta dirección en tu navegador (Chrome, Firefox, Safari, etc.)

## Funciones principales

### 1. Ver todos los clientes
- Al abrir la aplicación, verás la lista de todos tus clientes
- Cada cliente muestra su nombre y datos básicos

### 2. Agregar un nuevo cliente
1. Busca el botón "Agregar Cliente" o "+" 
2. Completa los datos:
   - Nombre (obligatorio)
   - Teléfono
   - Email
   - Empresa
3. Haz clic en "Guardar"

### 3. Ver detalles de un cliente
1. Haz clic sobre el nombre del cliente
2. Verás toda su información
3. Podrás ver sus notas
