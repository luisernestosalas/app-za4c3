# CRM con Clientes y Notas

Una aplicación web simple para la gestión de clientes y sus respectivas notas, desarrollada con tecnologías web estándar.

## 🚀 Demo

Puedes probar la aplicación en vivo aquí: [CRM Demo](https://app-za4c3-md8yj1kee-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ Gestión completa de clientes (crear, editar, eliminar)
- 📝 Sistema de notas asociadas a cada cliente
- 💾 Almacenamiento local en el navegador
- 📱 Interfaz responsive y fácil de usar
- 🔍 Búsqueda y filtrado de clientes
- 🎨 Diseño limpio y moderno

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura de la aplicación
- **CSS3** - Estilos y diseño responsive
- **JavaScript (Vanilla)** - Lógica de la aplicación
- **LocalStorage** - Persistencia de datos
